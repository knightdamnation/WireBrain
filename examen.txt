	GIT
	
	Cual es la diferencia entre git y GitHUB?
  Git es un sistema de control de revisiones, una herramienta para administrar el historial del código fuente. GitHub es un servicio de alojamiento para repositorios Git.
  
	En cuantos y cuales estados puede estar un archivo?
  Git tiene tres estados principales en los que se pueden encontrar tus archivos: confirmado (committed), modificado (modified), y preparado (staged).
  
	Para que sirve el comando "git add ." ?
  git add . 
  agrega todos los archivos y nuevos archivos en el directorio base y todos los subdirectorios al estado de staging para prepararse para el git commit con exception de git ignore
